# Patchwing – Essential Debug Tools

Patchwing is a lightweight, powerful developer tool designed to make **WordPress debugging** simple and effective for site administrators. Instead of digging through complicated configuration files, it provides clear, real-time data to help you resolve issues like the white screen of death or optimize site performance.

---

## 🚀 Features

### **System Dashboard**
- **High-Level Overview:** View WordPress, PHP, MySQL and cURL versions at a glance.
- **Environment Status:** Monitor GD Library, Multisite status and active/inactive plugin counts.
- **Server Health:** Live tracking of **CPU load** and **actual RAM usage** with visual status indicators.

### **Developer Insights**
- **Advanced PHP Info:** Detailed breakdown of `memory_limit`, `upload_max_filesize`, `max_execution_time` and active extensions.
- **Integrated Debug Log:** Monitor, filter and manage PHP error logs directly from the admin panel. Includes one-click "Clear Log" and "Enable/Disable" toggles.
- **Database Audit:** View all tables, storage engines and data/index lengths. Includes a built-in **engine migration tool**.

### **Performance Monitoring**
- **Metrics Tracking:** Monitor baseline load times and peak memory usage.
- **Theme Analysis:** Track specific load times for your active theme.
- **Visual Data:** Performance stats are visualized through interactive charts.

---

## 🛠 Installation

1. **Download:** Upload the `patchwing` folder to the `/wp-content/plugins/` directory.
2. **Activate:** Activate the plugin through the 'Plugins' menu in WordPress.
3. **Access:** Navigate to the 'Patchwing' menu in your WordPress Dashboard to start using the tools.
